This post shows how you could build an Imgur Android app using the latest Android architecture and stacks. 

Here are the stacks: 

- Architecture
	- Android MVVM
	- LiveData
	- Kotlin Coroutines
	- Hilt Dependency Injection

- Networking
	- Retrofit
	- Moshi - JSON parsing
	- Sandwich - handle APIResponse

- Data Layer
	- Room

- Imgur API
